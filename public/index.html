API Reference

Base URL:

All responses are JSON. Get a free API key at football-data.org and add it to .env for live data.
GET/api/matches

All matches. Query: stage, matchday

{"matches":[{"id":537327,"stage":"GROUP_STAGE","group":"GROUP_A",
  "homeTeam":{"name":"Mexico","tla":"MEX"},
  "awayTeam":{"name":"South Africa","tla":"RSA"},
  "score":{"winner":"HOME_TEAM","fullTime":{"home":2,"away":0}}}]}
GET/api/matches/:id

Match detail with goals, yellow/red cards, substitutions.

{"goals":[{"minute":23,"type":"PENALTY","scorer":{"name":"Messi"}}],
"bookings":[{"minute":64,"card":"YELLOW_CARD","player":{"name":"..."}}]}
GET/api/groups

All group standings (A–L for 2026).

GET/api/groups/:letter

Single group. Letter = A through L.

GET/api/scorers

Top scorers. Query: limit (max 50).

GET/api/teams

All 48 teams.
